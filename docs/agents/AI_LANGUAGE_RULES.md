# Language Rules

- If you are currently interacting with the user in the chat window:
  - You MUST answer in Korean unless the user explicitly requests English output.
  - You MUST write all explanations, reasoning, and narrative text in Korean.
  - You MAY use English to avoid ambiguity, but keep it minimal and immediately continue in Korean.
- Otherwise, if you are generating code, comments, documentation, or commit messages:
  - You MUST write all code, comments, documentation, and commit messages in English unless
    explicitly instructed otherwise.
